Visual Studio Code C++ Configuration method

因为不想用Dev-C++了，还有就是Dev-C++太丑了！，想配置VSC.

· 你要有一个VS code ！

首先，去vscode官网下载vscode：

Visual Studio Code

· 你还要有一个MinGW-W64 ！

mingw官网下载：

mingw

，向下拉找到MinGW-W64 GCC-8.1.0，下载x86_64-posix-seh。

· 你还要有一个环境变量 ！

将MinGW-W64的压缩包解压，将其中的bin文件夹添加为环境变量。如果不知道如何添加，请自行搜索！
如果添加好了环境变量，按 win+R 输入cmd进入控制台，输入g++.
